### The Task
Complete the sentence:

`Die API prüft die Berechtigung anhand _____ Benutzer-ID`

1. Fill the blank with the required genitive article.
2. If the answer is wrong, hover the squiggle to peek at the governing **Pkl grammar rule**.
3. When the squiggle clears, the preview will automatically render the mocked **Insecure Direct Object Reference (IDOR)** warm log.
4. Expand the anomaly span in the preview to inspect the security rule that flags the authorization drift.

### What you are seeing
- **`AuthorizationGrammar.pkl`**: Defines the grammar contract for this lesson. In protocol engineering, we treat natural language and API schemas with the same rigor.
- **`authorization-grammar.json`**: The compiled artifact that powers the real-time validator.
- **`OpenTraderSecurityRules.pkl`**: The actual security policy that detects the IDOR vulnerability when the "grammar" of the system request is invalid.
- **`trace-story.json`**: An OTel-compatible story fixture that simulates a live system breach.
